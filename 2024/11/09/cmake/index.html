cmakeжҳҜдёҖдёӘи·Ёе№іеҸ°зҡ„зј–иҜ‘е·Ҙе…·пјҢеҸҜд»Ҙеҝ«йҖҹзј–иҫ‘жһ„е»әйЎ№зӣ®зј–иҜ‘зҺҜеўғпјҢиҮӘеҠЁз”ҹжҲҗMakefile

еҢәеҲ«пјҡ

aux_source_directoryжҠҠеҪ“еүҚзӣ®еҪ•дёӢжүҖжңүж–Үд»¶е‘ҪеҗҚSRC_LIST

include_directoriesжҳҜе…ЁеұҖжҖ§зҡ„пјҢдёәж•ҙдёӘйЎ№зӣ®и®ҫзҪ®еҢ…еҗ«зӣ®еҪ•пјӣtarget_include_directoriesеҸӘдёәзү№е®ҡзӣ®ж Үж·»еҠ еҢ…еҗ«зӣ®еҪ•пјҢйҖҡеёёеңЁеӯҗзӣ®еҪ•зҡ„add_executableжҲ–add_libraryдҪҝз”Ё

  • PRIVATEпјҡеҢ…еҗ«зӣ®еҪ•еҸӘеҜ№жҢҮе®ҡзҡ„зӣ®ж Үжңүж•ҲпјҢдёҚдјҡдј йҖ’з»ҷдҫқиө–иҜҘзӣ®ж Үзҡ„е…¶д»–зӣ®ж ҮгҖӮ
  • PUBLICпјҡеҢ…еҗ«зӣ®еҪ•еҜ№жҢҮе®ҡзҡ„зӣ®ж Үжңүж•ҲпјҢе№¶дё”дјҡдј йҖ’з»ҷдҫқиө–иҜҘзӣ®ж Үзҡ„е…¶д»–зӣ®ж ҮгҖӮ
  • INTERFACEпјҡеҢ…еҗ«зӣ®еҪ•еҸӘеҜ№дҫқиө–иҜҘзӣ®ж Үзҡ„е…¶д»–зӣ®ж Үжңүж•ҲпјҢдёҚдјҡеҜ№жҢҮе®ҡзҡ„зӣ®ж Үжң¬иә«жңүж•ҲгҖӮ

cmake_minimum_required(VERSION 3.0)
project(chat)

# й…ҚзҪ®зј–иҜ‘йҖүйЎ№
set(CMAKE_CXX_FLAGS ${CMAKE_CXX_FLAGS} -g)

# й…ҚзҪ®жңҖз»Ҳзҡ„еҸҜжү§иЎҢж–Үд»¶иҫ“еҮәзҡ„и·Ҝеҫ„
set(EXECUTABLE_OUTPUT_PATH ${PROJECT_SOURCE_DIR}/bin)

# й…ҚзҪ®еӨҙж–Үд»¶зҡ„жҗңзҙўи·Ҝеҫ„
include_directories(${PROJECT_SOURCE_DIR}/include)

include_directories(${PROJECT_SOURCE_DIR}/lib)
link_directories(/usr/lib64/mysql)

# еҠ иҪҪеӯҗзӣ®еҪ•
add_subdirectory(src)

еҪ“еҢ…еҗ«еӨҡдёӘcmakeж–Үд»¶ж—¶пјҢеңЁжңҖй«ҳеұӮж·»еҠ е…ЁеұҖжҖ§зҡ„и®ҫзҪ®пјҢз„¶еҗҺadd_subdirectoryпјҢдёҖдёӘеҘҪзҡ„и®ҫи®ЎжҳҜеҸӘеҢ…еҗ«include_directories(${PROJECT_SOURCE_DIR}/include)пјҢеӨҙж–Үд»¶еҠ дёҠзӣ®еҪ•и·Ҝеҫ„#include вҖңserver/db/db.hвҖқпјҢиҝҷж ·еҸҜд»Ҙжңүж•ҲйҳІжӯўж–Үд»¶еҗҚеҶІзӘҒ

CMakeеёёз”Ёзҡ„йў„е®ҡд№үеҸҳйҮҸ

  • PROJECT_NAME : йҖҡиҝҮ project() жҢҮе®ҡйЎ№зӣ®еҗҚз§°
  • PROJECT_SOURCE_DIR : е·ҘзЁӢзҡ„ж №зӣ®еҪ•
  • PROJECT_BINARY_DIR : жү§иЎҢ cmake е‘Ҫд»Өзҡ„зӣ®еҪ•
  • CMAKE_CURRENT_SOURCE_DIR : еҪ“еүҚ CMakeList.txt ж–Үд»¶жүҖеңЁзҡ„зӣ®еҪ•
  • CMAKE_CURRENT_BINARY_DIR : зј–иҜ‘зӣ®еҪ•пјҢеҸҜдҪҝз”Ё add subdirectory жқҘдҝ®ж”№
  • EXECUTABLE_OUTPUT_PATH : дәҢиҝӣеҲ¶еҸҜжү§иЎҢж–Үд»¶иҫ“еҮәдҪҚзҪ®
  • LIBRARY_OUTPUT_PATH : еә“ж–Үд»¶иҫ“еҮәдҪҚзҪ®
  • BUILD_SHARED_LIBS : й»ҳи®Өзҡ„еә“зј–иҜ‘ж–№ејҸ ( shared жҲ– static ) пјҢй»ҳи®Өдёә static
  • CMAKE_C_FLAGS : и®ҫзҪ® C зј–иҜ‘йҖүйЎ№
  • CMAKE_CXX_FLAGS : и®ҫзҪ® C++ зј–иҜ‘йҖүйЎ№
  • CMAKE_CXX_FLAGS_DEBUG : и®ҫзҪ®зј–иҜ‘зұ»еһӢ Debug ж—¶зҡ„зј–иҜ‘йҖүйЎ№
  • CMAKE_CXX_FLAGS_RELEASE : и®ҫзҪ®зј–иҜ‘зұ»еһӢ Release ж—¶зҡ„зј–иҜ‘йҖүйЎ№
  • CMAKE_GENERATOR : зј–иҜ‘еҷЁеҗҚз§°
  • CMAKE_COMMAND : CMake еҸҜжү§иЎҢж–Үд»¶жң¬иә«зҡ„е…Ёи·Ҝеҫ„
  • CMAKE_BUILD_TYPE : е·ҘзЁӢзј–иҜ‘з”ҹжҲҗзҡ„зүҲжң¬пјҢ Debug / Release
